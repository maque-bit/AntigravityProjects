# 事業計画：MCP Insider — 自動エコシステム・パルス報告サービス（改訂版）

## 1. 事業概要
本事業は、急速に進化する MCP（Model Context Protocol）エコシステムの最新情報を自動で収集・整理し、開発者や AI 活用者向けに提供するサブスクリプション制のニュースレター、Web サービス、および通知プラグインです。単なる要約に留まらず、独自指標による分析を加えることで情報の専門性を担保します。

## 2. 収益モデル
- **サービス形態**: 有料サブスクリプション（月額 5ドル / 年額 50ドル）
- **ターゲット**: AI ツール開発者、エンタープライズ AI 導入担当者、MCP サーバー開発者
- **目標収益**: 月額 31.25ドル（初期目標）
  - 利益: 20ドル/月
  - 投資回収（ドメイン代等）: 1.25ドル/月
  - 運用費用（API 等）: 10ドル/月
- **必要顧客数**: 7名（月額プランの場合）

## 3. コスト構造
- **初期投資額**: 15ドル（独自ドメイン取得費用）
- **月間運用費**:
  - ホスティング: 0ドル（Vercel/GitHub Pages の無料枠を活用）
  - データ収集用 API (Brave Search/Gemini API): 5ドル（従量課金、初期は最小限）
  - 配信・決済手数料（Stripe 等）: 収益の約 5-10%（約 3ドル）
  - ストレージ・計算リザーブ: 2ドル
- **合計コスト**: 約 10ドル/月

## 4. 役割分担
### AI（Antigravity）の役割
- **情報収集**: GitHub, X, 公式ドキュメントからの自動スクレイピングと検索。
- **コンテンツ生成**:
  - **独自指標分析**: トレンドスコア（GitHubスター増加率）、実装難易度（コード複雑度分析）、セキュリティ信頼度。
  - **重要度の判定**: 要約、日本語への翻訳・解説記事作成。
- **Webサイト更新**: 要約記事の公開（静的サイトジェレーターの自動更新）。リードマグネット（無料特典）の常時更新。
- **配信最適化**: ニュースレター配信、および Slack/Discord 通知用データのドラフト作成。
- **コスト管理**: キャッシュ戦略による API 節約、予算上限の自動監視。

### 人間の役割
- **決済代行・法的責任**: Stripe アカウントの開設、強固な利用規約（免責事項）の承諾。
- **ドメイン管理**: ドメインの購入と DNS 設定。
- **プロモーション（存在証明）**:
  - **リードマグネット提供**: 「MCPサーバー100選」等の無料特典を通じたメール収集。
  - **コミュニティ貢献**: Qiita/Zenn 等での知見共有を通じた信頼性（専門家地位）の構築。
- **最終監修**: 配信内容のハルシネーション（事実誤認）チェック（1週間に10分程度）。

## 5. 投資回収計画
- **投資額**: 15ドル
- **回収期間**: 12ヶ月以内
- **回収方法**: 毎月の売上から 1.25ドルを原資に充当。
- **損益分岐点**: 月額ユーザー 4名時点で運用費をカバー、7名以降は純利益 20ドルを確保。

## 6. 導入予定の MCP サーバー
- 現時点では無料の標準サーバー（Brave Search 等）で完結。
- 将来的に効率化が必要な場合、GitHub 監視専用 MCP サーバーを導入（無料）。
- 有料 MCP が必要になった場合は、利益目標（20ドル）を維持したまま、価格に反映。

## 7. リーガル・リスク管理
- **免責事項の明記**: 「情報の正確性を保証せず、本情報による損害について責任を負わない」旨を明記。
- **AI 監修プロンプト**: コンテンツ生成時に事実確認を促す専用プロンプトを組み込み、ハルシネーションを抑制。
- **GDPR 準拠**: メールアドレス収集時の同意取得とプライバシーポリシーの整備。

## 8. 検証プラン
### 自動テスト
- データ収集スクリプトのモックテスト: `npm test` による収集エンジンの正常動作確認。
- インサイト整合性テスト: 独自指標（トレンドスコア等）の計算ロジック検証。

### 手動検証
- **リードマグネット収集テスト**: SNS 経由で無料特典が何名のユーザーに届くかを確認。
- **Stripe テスト**: テストモードによるサブスクリプション移行フローの確認。

## 9. ユーザーレビュー検討事項
> [!IMPORTANT]
> - リードマグネット（無料特典）を用意することで、SNSからの流入を有機的に有料購読へ繋げます。
> - 市場拡大局面では、法人向け高単価プラン（月額19ドル〜）への移行も視野に入れます。
