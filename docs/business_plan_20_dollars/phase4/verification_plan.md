# 動作検証計画：Phase 4

## 概要
開発した各コンポーネント（管理画面、フロントWeb、収集エンジン、分析エンジン）を Podman ローカル環境で起動し、動作確認が可能な状態にします。

## 起動・確認対象
| コンポーネント | URL / 確認方法 | ポート |
| :--- | :--- | :--- |
| **管理画面 (Admin)** | `http://localhost:3000` | 3000 |
| **フロントWeb (Web)** | `http://localhost:4321` | 4321 |
| **収集エンジン** | コンテナログ (`podman logs mcp-collector`) | - |
| **分析エンジン** | コンテナログ (`podman logs mcp-analyzer`) | - |

## 手順
1. **環境起動**: `podman-compose -f src/compose.yaml up -d` を実行。
2. **接続確認**: 各URLにブラウザでアクセス。
3. **ログ確認**: バックグラウンドエンジンの正常動作をログで確認。
